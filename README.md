# Windmill Breeze Theme 🌸

**Windmill Breeze** 是一款清新、灵动且功能丰富的 WordPress 个人作品集主题。它不仅仅是一个展示内容的网站，更是一个能够感知“天气”的互动空间。

## 🌟 核心亮点

### 1. 🌤️ 沉浸式天气交互
主题内置了天气系统，能够根据用户的地理位置获取实时天气（或通过点击天气卡片手动模拟）：
- **晴天 (Sunny)**：微风拂面，风车缓缓转动。
- **多云 (Cloudy)**：云朵飘动，风速增加。
- **雨天 (Rainy)**：雨滴落下，风车在风雨中疾驰。

### 2. 🇳🇱 动态荷兰风车
页面右下角驻守着一座精致的 CSS 绘制荷兰风车：
- **实时响应**：风车的转速会根据真实（或模拟）的风速数据自动调整。
- **数据可视化**：风车旁悬浮显示当前的风速（km/h）和风向。
- **智能防遮挡**：精心设计的层级和透明度，既是美丽的背景装饰，又不会干扰内容阅读。

### 3. 📱 极致的移动端体验
- 重新设计的移动端导航栏，支持横向滑动。
- 作品集和文章列表在手机上自动调整为单栏卡片布局。
- 优化的触控交互和视觉比例。

### 4. 🎨 现代化 UI 设计
- **作品集 (Portfolio)**：卡片式网格布局，悬停缩放特效。
- **简历预览**：支持在线预览 PDF 简历，无需下载即可查看。
- **深色模式**：内置日夜模式切换，呵护双眼。
- **磨砂玻璃质感**：在导航栏和浮层中使用 Backdrop Filter 营造现代感。

## 📂 目录结构

```text
windmill-breeze/
├── assets/             # 静态资源 (CSS, JS)
├── archive-portfolio.php # 作品集归档页
├── front-page.php      # 首页模板
├── functions.php       # 主题核心逻辑
├── header.php          # 全局头部
├── footer.php          # 全局底部 (包含风车动画)
└── style.css           # 主样式表
```

## 🚀 开始使用

1. 下载本仓库代码。
2. 将 `windmill-breeze` 文件夹放入 WordPress 的 `wp-content/themes/` 目录。
3. 登录 WordPress 后台，在 **外观 > 主题** 中启用 **Windmill Breeze**。
4. 访问首页，享受微风！

## 📝 自定义配置

你可以在 WordPress 的 **外观 > 自定义** 中设置：
- 社交媒体链接
- 简历文件路径
- 首页座右铭
